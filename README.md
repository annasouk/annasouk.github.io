# annasouk.github.io

Personal portfolio site for Anna Soukhovei — robotics researcher at CU Boulder.

## Local Preview

Open `index.html` in a browser, or serve it locally:

```bash
python3 -m http.server 8000
```

Then visit [http://localhost:8000](http://localhost:8000).

## Deployment

This site is served by GitHub Pages from the repository root. Any push to `main` will automatically update [https://annasouk.github.io](https://annasouk.github.io).
